VSync is a full-stack college companion mobile platform designed to centralize student academic and campus information into a single application.

##  Features
- Timetable management
- Attendance percentage tracking
- GPA & CGPA monitoring
- Faculty cabin detail retrieval (backend integrated)
- Modular UI design

##  Tech Stack
Flutter, Dart, REST API, Backend Server, Database, Git

##  Architecture
Flutter frontend communicates with backend APIs to fetch and display faculty cabin information dynamically.
